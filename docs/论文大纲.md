# 基于多源地理数据与群体智能优化的输电线路路径规划研究

## 摘要 (200字)
- 问题：复杂地形下输电线路路径规划的多因素约束优化
- 方法：RF/CNN 成本建模 + A\*/IPSO-SA/DBO 路径搜索 + 7项质量门控
- 实验：台湾地区10条真实线路验证, CNN-Hausdorff 4595m
- 创新：DBO首次引入输电线路规划, CNN像素级成本预测

## 1. 引言
- 1.1 研究背景：山区输电线路规划的工程挑战
- 1.2 现有方法综述：GIS选线、元启发式、深度学习
- 1.3 研究目标与贡献

## 2. 数据与预处理
- 2.1 研究区域与数据源（台湾, DEM 30m, OSM, 输电线1795条）
- 2.2 18维特征体系（7地形+4距离场+土地利用+建筑密度+3风险代理+铁路）
- 2.3 硬/软双层约束掩膜（坡度≤45°, 高程≤3000m, 保护区200m缓冲等）
- 2.4 90m统一栅格对齐

## 3. 成本表面建模
- 3.1 伪标签生成（领域知识加权公式, 7因子, 权重0.02-0.65）
- 3.2 随机森林回归（R²>0.97, 特征重要性分析）
- 3.3 U-Net像素级回归（18通道输入→1通道成本输出, 7.8M参数）
- 3.4 成本表面融合（走廊偏好corridor_bonus + 软约束）

## 4. 路径规划算法
- 4.1 A\*搜索（Octile启发式, 8邻域, 全局最优保证）
- 4.2 IPSO-SA（PSO惯性递减0.9→0.4 + SA退火exp(-Δf/T)）
- 4.3 DBO蜣螂优化（滚球/跳舞/育雏/觅食/偷窃, 五行为模型, 2022）
- 4.4 路径平滑与7项质量门控

## 5. 实验设计
- 5.1 10条标准测试线路（覆盖台湾全岛, 32-110km, 不同地形）
- 5.2 评价指标（Hausdorff距离, 500m重叠率, 长度误差, 质量通过率）
- 5.3 对比方案（RF vs CNN成本表面, A\* vs IPSO-SA vs DBO）
- 5.4 统计检验（配对t检验, Cohen's d效应量）

## 6. 实验结果
- 6.1 RF成本表面结果（Hausdorff 6227m, 10/10通过）
- 6.2 CNN成本表面结果（Hausdorff 4595m, ↓21%）
- 6.3 三算法对比（IPSO-SA最优, DBO分化）
- 6.4 统计显著性（Cohen's d=0.22, n=10）
- 6.5 消融实验（corridor_bonus有效性, DEM分辨率影响）

## 7. 讨论
- 7.1 CNN空间上下文 vs RF逐像素预测
- 7.2 群体智能在确定性成本表面的适用性
- 7.3 IPSO-SA与DBO的差异化分析
- 7.4 伪标签权重的局限性

## 8. 结论
- 8.1 主要贡献
- 8.2 局限性
- 8.3 未来工作
