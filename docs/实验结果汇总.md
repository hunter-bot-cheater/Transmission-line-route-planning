## 关键发现

1. **CNN成本表面显著提升路径质量**: IPSO-SA在CNN上Hausdorff↓21% vs RF
2. **IPSO-SA在CNN上为最佳算法**: Hausdorff 4,595m, 重叠率30.8%
3. **DBO在CNN上表现分化**: 走出不同于IPSO-SA的路径, 但质量通过率低(2/10)
4. **corridor_bonus验证有效**: 削弱后Hausdorff恶化, 证明原有参数合理
5. **12.5m→30m DEM是关键**: 原生分辨率计算地形因子, Hausdorff改善40%+
